# Music Player — Python

A feature-rich desktop music player built with Python, featuring playlist management, audio controls, and a clean graphical interface.

## Features

- **Play / Pause / Stop / Next / Previous** controls
- **Import music** from your local file system
- **Playlist management** — add, remove, reorder tracks
- **Track info display** — file name and playback status
- **Volume control**
- Lightweight, no external media dependencies beyond Pygame

## Tech Stack

| Library | Purpose |
|---|---|
| Tkinter | GUI framework |
| Pygame | Audio playback engine |
| PIL / Pillow | Image handling for UI assets |
| os / pathlib | File system navigation |

## Getting Started

```bash
git clone https://github.com/shree302001/Music-Player-Python.git
cd Music-Player-Python

pip install pygame pillow

python music.py
```

**Supported formats:** MP3, WAV, OGG

## Requirements

- Python 3.7+
- Pygame 2.x
- Pillow
